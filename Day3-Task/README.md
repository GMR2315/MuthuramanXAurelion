
# Aurelion Backend Day - 3

Performing CRUD operations on Student database using API's using Prisma. 


## Tech Stack

Node.js and Express


## Installation

Install packages with npm

```bash
  npm install
```

For development
```bash
  npm run dev
```

For production

```bash
  npm start
```
